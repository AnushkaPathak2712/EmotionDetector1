# Emotion Detector

A web application that detects emotions from text using IBM Watson NLP API.

## Features

- Analyze emotions from text input
- Display emotion scores (anger, disgust, fear, joy, sadness)
- Identify dominant emotion
- Beautiful, responsive UI
- Error handling for invalid input
- Unit tests for all major functionality
